# 项目简介 java917


## 技术栈

- 前端：Vue3、Element UI、Axios、Vuex、Router
- 后端：Node.js、MySQL、JWT

## 功能模块

### 前台

- 新闻及产品预览

### 后台

- 用户管理
- 新闻管理
- 产品管理
- 动态粒子效果展示

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：现代浏览器环境
- 后端：Node.js环境，MySQL数据库

## 部署步骤

1. 准备Node.js运行环境
2. 配置MySQL数据库
3. 部署后端服务
4. 部署前端项目

## 核心亮点

- 使用Vue3和Element UI构建了友好的用户界面。
- 通过JWT进行用户鉴权，保障系统安全。
- 前后端分离，便于维护和扩展。
